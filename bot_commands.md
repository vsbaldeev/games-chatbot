## Подбор игр

### `/multiplayer`
Подбирает одну PS5 или PS5/PC игру с онлайн-кооперативом или мультиплеером из IGDB.

- Проверяет кросплей с PC по платформам в IGDB
- Получает цену в TRY и прямую ссылку на турецкий PS Store через psdeals.net
- Дедупликация по чату — одна и та же игра не предлагается дважды подряд

Формат ответа:
```
🎮 Название игры
Жанр: ...
Игроков онлайн: до N
Кросплей с PC: Да / Нет / нет данных
Цена в TRY: ...
Краткое описание.
🛒 https://store.playstation.com/tr-tr/...
```

### `/singleplayer`
Подбирает одну одиночную игру для PS5 (рейтинг IGDB ≥ 75, без мультиплеерных режимов).

- Получает цену в TRY и ссылку на турецкий PS Store
- Дедупликация по чату

Формат ответа:
```
🎮 Название игры
Жанр: ...
Цена в TRY: ...
Краткое описание.
🛒 https://store.playstation.com/tr-tr/...
```

## D&D-игры

Все три D&D-команды работают только в группах (`ChatType.GROUPS`). Сначала открывается лобби, минимум 3 игрока. Если в чате зарегистрировано меньше 3 участников, бот заполняет недостающий слот NPC-игроком.

В одном чате одновременно может идти только одна игра.

### Фаза лобби

Бот отправляет сообщение с двумя инлайн-кнопками:

```
⚔️ D&D Приключение

Набирается отряд для опасного похода!
Нужно минимум 3 игрока.

Отряд (1) — нужно ещё 2:
• @caller

Лобби закроется через 5 минут.

[⚔️ Присоединиться]  [▶️ Начать (1/3)]
```

- Присоединиться может любой участник чата.
- Начать может любой, когда набралось 3+ игрока.
- Лобби закрывается автоматически через **5 минут**, если никто не нажал «Начать».

### Ход раунда

В каждом раунде:
1. LLM генерирует сценарий и 4 варианта действий.
2. У каждого игрока **45 секунд**, чтобы выбрать действие через инлайн-кнопки.
3. Бот бросает `d20` за каждого и передаёт результаты LLM.
4. LLM рассказывает, что произошло, с учётом броска (1 — критический провал, 20 — критический успех).

Каждый раунд — отдельное сообщение в чате; результаты накапливаются в истории переписки.

---

### `/dnd_pvp` — PvP (1 раунд)
Один раунд, все против всех. Побеждает тот, у кого лучший бросок.

### `/dnd_coop` — Кооп против босса (2 раунда)
Все игроки вместе сражаются против одного абсурдного босса-NPC.

- HP босса = `random.randint(N × 15, N × 20)`, где N — количество игроков
- Броски всех игроков суммируются как урон боссу за раунд
- Победа, если `boss_hp ≤ 0` после 2-го раунда (~70% вероятность победы для 3 игроков)
- Имя босса генерирует LLM в первом раунде и не меняется

### `/dnd3` — Великое Ограбление (3 раунда)
Три фазы ограбления: проникновение → дело → побег.

| Раунд | Фаза | Задача |
|-------|------|--------|
| 1 | Проникновение | Попасть внутрь незаметно |
| 2 | Дело | Добраться до цели и взять трофей |
| 3 | Побег | Уйти от погони |

Действия — только стелс и хитрость, лобовые атаки исключены. LLM оценивает выбранный трюк каждого игрока в контексте его броска d20.

## Социальные игры

### `/duel`
Запускает эмодзи-дуэль между двумя случайными участниками чата (вызывающий исключается, если зарегистрировано 3+ участников).

- Естественный триггер: напишите «хочу дуэль с @vasya», чтобы начать целевую дуэль без команды
- Первый, кто нажмёт 🔫, побеждает
- Таймаут 5 минут; истёкшая дуэль отменяется автоматически

### `/ruletka`
Русская рулетка по требованию. Нужно минимум 2 зарегистрированных участника.

- Бот случайно выбирает участника и стреляет: 50% попадание, 50% промах
- Ежедневная автоматическая рулетка запускается в **21:00 МСК**
- Выживания отслеживаются и дают достижения (до 50 выживаний)

### `/prozharka`
Прожаривает случайного участника чата.

- LLM строит прожарку на основе последних сообщений участника
- Стиль: коротко (≤ 2 предложений), саркастичный стендап-комик
- 10% шанс тёплого поддерживающего сообщения вместо прожарки

Автотриггер: если пользователь дважды подряд отвечает боту оскорблениями, бот прожаривает его автоматически.

## Статистика

### `/achievements`
Показывает последние 3 заработанных достижения и общее количество.

Достижения выдаются за:
- Активность в чате, реакции, стикеры, голосовые/видео/фото-сообщения
- Ночные сообщения, длинные сообщения, ссылки
- Выживания в рулетке, победы в дуэлях, прожарки
- Победу или участие в D&D-играх

### `/top`
Топ-3 чата по количеству достижений с медальными эмодзи.

## Взаимодействие без команд

| Триггер | Поведение |
|---------|-----------|
| Упоминание через `@` или ответ на сообщение бота | Отвечает на любой вопрос; использует инструменты IGDB и Steam для вопросов об играх |
| Голосовое сообщение или кружок | 25% шанс транскрибировать и прокомментировать (пересланные сообщения и боты пропускаются) |
| Оскорбительный ответ боту (×2) | Автоматическая прожарка |
| «хочу дуэль с @user» | Запускает целевую дуэль без `/duel` |
