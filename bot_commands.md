## Game suggestions

### `/multiplayer`
Picks one PS5 or PS5/PC co-op or online multiplayer game from IGDB.

- Checks PC crossplay by looking up platform data on IGDB
- Fetches TRY price and a direct Turkish PS Store link via psdeals.net
- Deduplicates per chat — the same game is never suggested twice in a row

Output format:
```
🎮 Game Name
Жанр: ...
Игроков онлайн: до N
Кросплей с PC: Да / Нет / нет данных
Цена в TRY: ...
One-line description.
🛒 https://store.playstation.com/tr-tr/...
```

### `/singleplayer`
Picks one PS5 single-player game (IGDB rating ≥ 75, no multiplayer modes).

- Fetches TRY price and Turkish PS Store link
- Deduplicates per chat

Output format:
```
🎮 Game Name
Жанр: ...
Цена в TRY: ...
One-line description.
🛒 https://store.playstation.com/tr-tr/...
```

## D&D party games

All three D&D commands require **groups only** (`ChatType.GROUPS`). A lobby is opened first; minimum 3 players required. If the chat has fewer than 3 registered members, the bot fills the missing slot with an NPC player.

Only one D&D game can run per chat at a time.

### Lobby flow

The bot sends a message with two inline buttons:

```
⚔️ D&D Приключение

Набирается отряд для опасного похода!
Нужно минимум 3 игрока.

Отряд (1) — нужно ещё 2:
• @caller

Лобби закроется через 5 минут.

[⚔️ Присоединиться]  [▶️ Начать (1/3)]
```

- Any chat member can join with the first button.
- Any member can start once 3+ players have joined.
- The lobby auto-closes after **5 minutes** if nobody starts.

### Round flow

Each round:
1. LLM generates a scenario and 4 action options.
2. Players have **45 seconds** to pick an action via inline buttons.
3. The bot rolls `d20` for each player and sends results to the LLM.
4. LLM narrates what happened, weighted by each player's roll (1 = critical fail, 20 = critical success).

Each round is a new message in the chat; results accumulate in the chat history.

---

### `/dnd_pvp` — PvP (1 round)
One round, everyone against each other. Best roll wins.

### `/dnd_coop` — Co-op vs Boss (2 rounds)
All players fight a single absurd boss NPC together.

- Boss HP = `random.randint(N × 15, N × 20)` where N = player count
- All players' rolls are summed as damage dealt to the boss each round
- Victory if `boss_hp ≤ 0` after round 2 (~70% win probability for 3 players)
- Boss name is generated by the LLM in round 1 and stays fixed

### `/dnd3` — The Great Heist (3 rounds)
Three-phase heist: infiltration → the job → escape.

Each phase has a distinct theme:

| Round | Phase | Goal |
|-------|-------|------|
| 1 | Проникновение | Get inside undetected |
| 2 | Дело | Reach the target and grab the loot |
| 3 | Побег | Shake the pursuit |

Actions are limited to stealth and trickery — no frontal attacks. The LLM evaluates each player's chosen trick in the context of their d20 roll.

## Social games

### `/duel`
Starts an emoji duel between two random chat members (caller is excluded when 3+ members are registered).

- Natural language trigger: write «хочу дуэль с @vasya» to start a targeted duel without the command
- First player to tap 🔫 wins
- 5-minute timeout; expired duels are cancelled automatically

### `/ruletka`
On-demand Russian roulette. Requires at least 2 registered members.

- The bot randomly picks one member, then fires with a 50% hit / 50% miss outcome
- A daily automatic roulette also runs at **21:00 MSK**
- Survivals are tracked and award achievements (up to 50 survivals)

### `/prozharka`
Roasts a randomly chosen chat member.

- The LLM bases the roast on the member's recent messages
- Style: short (≤ 2 sentences), sarcastic stand-up comedian
- 10% chance of a warm supportive message instead of a roast

Auto-trigger: if a user replies to the bot with offensive words twice in a row, the bot roasts them automatically.

## Stats

### `/achievements`
Shows your last 3 earned achievements with the total count.

Achievements are granted for:
- Message activity, reactions, stickers, voice/video/photo messages
- Night messages, long messages, links
- Roulette survivals, duel wins, being roasted
- Winning or participating in D&D games

### `/top`
Top-3 leaderboard for the current chat, ranked by total achievement count.

## Non-command interactions

| Trigger | Behaviour |
|---------|-----------|
| `@mention` or reply to the bot | Answers any question; uses IGDB and Steam tools for game queries |
| Voice message or video note | 25% chance the bot transcribes and comments (skips forwarded messages and other bots) |
| Offensive reply to bot (×2) | Auto-roast triggered |
| «хочу дуэль с @user» | Starts a targeted duel without `/duel` |
